## 📊 作品简介

本作品对中国 31 个省级行政区（不含港澳台）的**森林覆盖率**与**湿地覆盖率**进行横向对比，结合地理区位、地形地貌、气候条件等多维因素，揭示中国生态资源分布的**东南—西北分异规律**：

- 🥇 **森林覆盖率前三**：福建 66.80% · 江西 61.20% · 广西 60.20%
- 💧 **湿地覆盖率前三**：上海 50.74% · 天津 14.44% · 黑龙江 13.62%
- 🏔️ **湿地总面积前三**：西藏 523.2 万公顷 · 黑龙江 431.5 万公顷 · 青海 412.6 万公顷

## 📚 数据来源

- **森林覆盖率**：第九次全国森林资源清查（截至 2024 年）
- **湿地面积**：中国首次湿地资源调查（1995-2003），国家统计局发布
- **省辖区面积**：公开地理数据

> 湿地覆盖率 = 湿地面积 ÷ 省辖区面积
